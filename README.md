# Сценарии уроков о культуре России

Одностраничное приложение на Next.js (App Router) и Tailwind CSS, которое помогает учителям подбирать сценарии уроков о культуре России по теме, возрасту, времени и цифровым инструментам.

## Технологии

- **Next.js** (App Router)
- **React** (`useState` для фильтрации)
- **Tailwind CSS**
- **lucide-react** (иконки)

## Как запустить проект

1. Установите Node.js (рекомендуется LTS-версия).

2. Перейдите в папку проекта и установите зависимости:

   ```bash
   cd russian-culture-lessons
   npm install
   ```

3. Запустите сервер разработки:

   ```bash
   npm run dev
   ```

4. Откройте в браузере страницу [http://localhost:3000](http://localhost:3000).

## Реализованные функции

- **Фильтрация сценариев** — 4 выпадающих списка: Тема, Возраст, Время, Инструменты. Фильтрация через `useState`. Подбор по точному совпадению темы, возраста и времени, по частичному совпадению инструмента. Если совпадений нет — сообщение «Попробуйте изменить настройки».

- **Карточки сценариев** — сетка карточек с названием, темой, возрастом, временем и краткой целью. Адаптивный дизайн.

- **Модальное окно «Подробнее»** — при клике отображаются все 7 полей сценария: Название, Возраст, Цель, Как начать, Основная часть, Рефлексия, Главный вывод, плюс список цифровых инструментов.

- **UI/UX** — светлый фон, сдержанные цвета, системные шрифты, адаптивная вёрстка для мобильных устройств.
